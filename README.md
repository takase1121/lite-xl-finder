## lite-xl-finder

a fuzzy finder for lite-xl, inspired by [fzf][1].

> **Note: this is a work in progress.**

![image][2]

### Install
```sh
git clone https://github.com/takase1121/lite-xl-finder.git finder
```
In your plugin directory.
